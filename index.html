Ayer se hablo sobre el cifrado ssh este cifrado costa de dos partes una llave publica y una llave privada este sifrado
permite compartir información cifrada por internet lo unico que necesitas es compartir tu llave publica con otros para
que sifren el contenido que quieran enviarte despues de que te lo envian este es sifrado junto con la llave privada
ojo nunca des tu LLAVE PRIVADA ya que si la compartes pueden cifrar cualquier información que envies por internet

Las llaves publicas tambien las puedes añadir a github en las opciones de settings y llaves ssh "keys ssh" esto permite subir
información a un repositorio de manera mas segura ya que no compartes tus contraseñas por internet y evitas que tambien que escribirla
ya que con las llaves en el repositorio este automaticamente subira los archivos a este repositorio.

Tambien aprendimos de como hacer tags y esto se hace con el $ git tag -a nombre del tag + #commit al que se le quiera agregar
el tag para borrar se usa $ tag -d + nombre del tag para eliminar. para subir los tag a un repositorio en linea usas $git push origin --tags
y para borrar del repositorio remoto se usa el $git push origin :refs/tags/nombre del tag ojo esto hay que hacerlo luego de eliinar y
subir el tag normalmente.
